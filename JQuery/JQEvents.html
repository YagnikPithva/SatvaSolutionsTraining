jQuery Event Demonstration

1. Click Event

Click the box to change its text:

Click me

      $("#clickBox").click(function() {
        $(this).text("You clicked me!");
      });
2. Hover Event

Hover over the box to change its color:

Hover me

      $("#hoverBox").hover(
        function() { // mouse enter
          $(this).css("background-color", "#2ecc71");
        },
        function() { // mouse leave
          $(this).css("background-color", "#e74c3c");
        }
      );
3. Keydown Event

Type something in the input field to trigger the keydown event:


      $("#keydownInput").keydown(function(event) {
        alert("Key down: " + event.key);
      });
4. Keyup Event

Type something in the input field and see the keyup event in action:


      $("#keyupInput").keyup(function(event) {
        console.log("Key up: " + event.key);
      });
5. Focus Event

Click or tab into the input field to trigger the focus event:


      $("#focusInput").focus(function() {
        $(this).css("border", "2px solid #3498db");
      });
