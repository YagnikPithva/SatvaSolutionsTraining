2. jQuery Selectors

jQuery uses CSS-like selectors to target HTML elements. Here are some of the most common selectors:

By ID Selector

By using #id, we can select elements by their ID attribute.

By Class Selector

By using .class, we can select all elements with a specific class.

By Element Selector

By using element, we can select all instances of an element (e.g., <p>).
